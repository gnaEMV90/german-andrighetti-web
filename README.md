# German Andrighetti Web

Sitio institucional para servicios de desarrollo de software a medida.

## Stack

- Vite
- React
- TypeScript
- CSS puro
- Cloudflare Pages

## Uso local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Cloudflare Pages

- Production branch: `main`
- Build command: `npm run build`
- Build output directory: `dist`
